# rabbithole

# Installation for Ubuntu..

(1) Download..

Open Terminal (CTRL+ALT+T) and run..

	$ wget --no-check-certificate https://github.com/alectramell/rabbithole/raw/master/rabbithole-1.0.deb

(2) Install..

Now run..

	$ sudo dpkg -i rabbithole-1.0.deb

(3) Add to Dashboard..

Finally Run..

	$ rabbithole

RabbitHole will show up in your file browser, drag the icon to you dashboard to place it, and there you are! Now start tagging some Developers! ;)
